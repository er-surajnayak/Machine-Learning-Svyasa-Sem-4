Data Types
in Machine Learning

ML models only understand numbers. Everything else — text, images, audio, categories — must be converted. But first you need to know what type of data you're dealing with.

01 · The foundation

Why data types matter

ML doesn't understand emotions, words, or "good vibes." It only understands numbers. So the type of data you feed matters a lot — different types behave differently, and treating them incorrectly gives wrong results.

The core problem: If Branch is encoded as CSE=1, ECE=2, MECH=3 — the model thinks MECH > CSE. But there's no ranking between branches! Understanding data type = understanding how to prepare data correctly.

02 · Type 1

Numerical data

The simplest type. Numbers. Actual measurable values. These have meaning in magnitude — 10 study hours is actually double 5 hours.

Examples: Age, Salary, CGPA, Temperature, Study hours, Height, Weight
Discrete

Whole numbers — you count them

Internships0, 1, 2, 3…
Siblings0, 1, 2…
Projects1, 2, 3…

You can't have 2.3 internships. Values are integer-only.

Continuous

Any value in a range — you measure them

Height170.5 cm
Weight65.2 kg
Temperature36.6°C

Decimal values are meaningful. Can be infinitely precise.

Quick trick — Discrete vs Continuous: Can I break it into smaller decimals meaningfully? YES → Continuous. NO → Discrete.
Height = 170.2 cm ✓ Continuous. Internships = 2.3? ✗ Doesn't make sense. Discrete.
03 · Type 2

Categorical data

Labels or names. Not numbers. No mathematical meaning between them — no ordering, no ranking.

Examples
BranchCSE, ECE, MECH
CityBengaluru, Delhi, Mumbai
Blood groupA, B, AB, O
GenderM, F, Other
The danger

If you encode: CSE=1, ECE=2, MECH=3 — the model thinks:

MECH (3) > CSE (1)?
WRONG! No such ranking exists.

Solution: Use one-hot encoding (covered in Topic 4).

04 · Type 3

Ordinal data

Special type of categorical data. There is order. But the exact distance between values is not measurable.

Key distinction: Ordinal = order exists, but gaps are unequal or unknown. Is "High" communication twice as good as "Medium"? Not necessarily. We just know High > Medium > Low.
Examples with order
CommunicationLow < Medium < High
T-shirt sizeS < M < L < XL
Star rating1 < 2 < 3 < 4 < 5
GradeC < B < A < A+
How to encode

Use label encoding that preserves order:

Low → 0
Medium → 1
High → 2

Unlike categorical, order-preserving encoding is valid here.

05 · Beyond tables

Text, image, and audio data

ML doesn't only work with tables. These data types are everywhere in modern AI — and they all become numbers behind the scenes.

Text data

Words → numbers

Tweets, emails, reviews, chat. The model doesn't "read" — it converts text to numerical vectors. Used in spam detection, sentiment analysis, chatbots.

Image data

Pixels → matrices

Images look visual to us. For ML? They're just matrices of pixel values (0–255). A 28×28 image = 784 numbers. Used in face recognition, X-ray analysis.

Audio data

Sound → frequency features

Voice assistants don't "hear" words. Audio → waveforms → frequencies → numerical features. Everything becomes numbers.

The big realization: Even text, images, and sound must be converted into numeric form. That's why data preprocessing is such a big deal — ML models only speak the language of numbers.
06 · Memory tricks

Simple ways to remember

Identity vs Quantity test

Ask yourself: Does this value answer "What type is this?" → Categorical. Does it answer "How much?" → Numerical.

Branch → "What type?" → Categorical. Salary → "How much?" → Numerical.

The decimal test

Can I have 2.5 of this meaningfully?

Internships = 2.5? ✗ → Discrete. Height = 170.5? ✓ → Continuous. Rating = 4.5? ✓ → Can be continuous.

07 · Check your understanding

Quick quiz

1. Blood group (A, B, AB, O) is which type of data?
A. Categorical ✓
B. Ordinal
C. Continuous Numerical
D. Discrete Numerical
Blood group is categorical — it's a label/name with no order and no numerical meaning. A is not greater than B. ✓
2. Customer satisfaction rating (1 star, 2 stars, 3 stars, 4 stars, 5 stars) is which type?
A. Categorical
B. Ordinal ✓
C. Continuous Numerical
D. Discrete Numerical
Ordinal — there IS an order (5 > 4 > 3 > 2 > 1) but the distance between stars is not uniform. Is the gap between 1★ and 2★ the same as 4★ and 5★? Not necessarily. ✓
3. Number of questions attempted in an exam is which type?
A. Categorical
B. Ordinal
C. Discrete Numerical ✓
D. Continuous Numerical
Discrete Numerical — you count questions (0, 1, 2, 3…). You can't attempt 2.7 questions. It's a whole number that represents quantity. ✓
